# META ADS GODMODE

Ein Claude-Code-Plugin, das Meta-Werbung (Facebook + Instagram) fuer
E-Commerce auf Weltklasse-Niveau plant und steuert — Kampagnen-Struktur,
Tracking (Pixel + Conversions API), Creative-Strategie, Budget-Skalierung,
KPI-Steuerung und Schweiz-Compliance (PBV, DSG).

## Was ist drin

| Baustein | Zweck |
|---|---|
| Skill `meta-ads` | Betriebsprotokoll in 5 Phasen: Fundament (Tracking + Break-even) → Kampagnen-Framework → Creative-Strategie → Budget/Messung/Skalierung → Schweiz & Compliance. Mit Detail-Playbook (`references/playbook.md`): Blueprints, Angle-Bibliothek, Copy-Geruest, KPI-Diagnose-Kette, Launch-Checkliste. |
| Command `/meta-ads <Aufgabe>` | Kompletter Durchlauf fuer eine konkrete Ads-Aufgabe — Ergebnis ist ein copy-paste-fertiger Kampagnen-Blueprint mit Copy, Creative-Briefs und Mess-Plan. |

## Harmonisierung mit den anderen Bausteinen

- **shopify-godmode** — Landingpages, CRO, Verkaufspsychologie und
  Creative-Produktion (inkl. Higgsfield); `meta-ads` uebernimmt Media
  Buying, Kampagnen-Logik und Skalierung.
- **ultra-enterprise-os** — orchestriert grosse Mehrteiler (z.B. Launch =
  Store + Ads + E-Mail); `meta-ads` liefert darin den Ads-Teil.

## Ehrlichkeits-Doktrin

Keine erfundenen Benchmarks, keine ROAS-Versprechen. Die Skill liefert
Blueprints und Entscheidungsregeln — geschaltet und gemessen wird im
echten Ads Manager.

## Installation

Wird ueber `nate-marketplace` (dieses Repo) geladen und ist in
`.claude/settings.json` aktiviert.
