3 · KURZBEFEHL AUF DEM IPHONE ANLEGEN

  1. App Kurzbefehle öffnen → + (neuer Kurzbefehl).
  2. Aktion „Text diktieren" hinzufügen (das sagst du Siri).
  3. Aktion „Inhalte von URL abrufen" hinzufügen:
    • URL: die Adresse von links (Punkt 2) — aber das TEXT_HIER am Ende löschen.
    • Auf Methode → POST.
    • Anfragetext → JSON → Feld text = Variable „Diktierter Text".
  4. Aktion „Text vorlesen" hinzufügen → Variable = Ergebnis der URL.
  5. Oben Namen geben: „JARVIS". Fertig ✓
Jetzt: „Hey Siri, JARVIS" → sag deinen Befehl → Siri liest JARVIS' Antwort vor.

BEISPIEL-BEFEHLE // WAS DU SAGEN KANNST

„Öffne YouTube" · „Spiel Lofi Musik auf YouTube" · „Logge dich bei Instagram ein" · „Such nach dem Wetter morgen" · „Was ist die Hauptstadt von Japan" · „Mach einen Screenshot"
EHRLICH: Der Kurzbefehl schickt deinen gesprochenen Text an JARVIS und liest die Antwort vor. JARVIS läuft weiter dort, wo der Server läuft (PC / Tailscale / Cloud) — nur so weit reicht sein Zugriff. Damit JARVIS unterwegs erreichbar ist, muss der Server von außen erreichbar sein (Tailscale oder Cloud). Ohne Token kann jeder mit der URL JARVIS steuern — setz darum ein Token, sobald JARVIS aus dem Netz erreichbar ist.
